## EAIoT framework
clone as below:
```bash
git clone https://github.com/EAIoTIR/EAIoT.git
```
to clone all submodules:
```bash
git submodule update --init --recursive
```
or use git submodule update --init for each repository separately

این پروژه با حمایت معاونت علمی در پردیس دانشکده های فنی دانشگاه تهران آزمایشگاه سیستم های نهفته چند هسته ای توسعه داده شده است و برای عموم به صورت متن باز قرار داده شده است.
